## Connect to database
Sử dụng 1 phần mềm kết nối đến database bất kì với credentials sau:
- host: localhost
- port: 54321 (nếu dùng postgres docker) và 5432 nếu dùng postgres ở máy
- db name: ecommerce_integrations
- user: postgres
- password: secret
